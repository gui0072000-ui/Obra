# ObraManager

Sistema de gestão de obras com cronograma Gantt, diário de obras e kanban.

## Estrutura do repositório

```
/
├── index.html      ← aplicação completa (tudo em um arquivo)
├── vercel.json     ← configuração do Vercel
└── README.md
```

## Deploy no Vercel

1. Faça push destes arquivos para a raiz do repositório
2. No Vercel, importe o repositório normalmente
3. Não é necessário nenhuma configuração de build

## Firebase

Ao abrir o site, clique no banner amarelo e cole seu firebaseConfig.
